## 第2条（業務内容）

### 2.1 人事評価システム
乙は、甲の人事評価業務を支援するWebシステムを開発・保守する。主な機能は以下の通り：

1. **ダッシュボード**: 全体の評価状況を俯瞰
2. **評価入力**: 部署別評価項目の入力・編集
3. **評価履歴**: 過去の評価データの管理
4. **部門比較**: 6角形レーダーチャート + 表形式での可視化
5. **マネージャー評価**: 評価対象者の一覧と詳細評価
6. **AI分析機能**: 評価傾向の分析（将来実装予定）
7. **分析レポート**: 評価分布の可視化
8. **管理機能**: 従業員マスター、監査ログ、権限管理
9. **Excel互換エクスポート**: 評価データのExcel出力

**開発状況**:
本システムは2025年9月25日の打ち合わせ時点において、Vercel環境にて全9機能のデモを実施済みである。残作業は実データ投入および本番環境デプロイのみである。

### 2.2 営業マン育成システム（共同開発・投資案件）
乙は、甲と共同で営業マン育成を支援するシステムを開発する。

**目的**: 「どのようにすれば会社の理想とする営業マンに育てられるか？」というテーマのもと、営業マン育成プロセスの標準化・体系化をAI技術を活用してシステム化する。

**位置づけ**: 本件は共同開発・投資案件とし、甲が投資として開発費を支払い、乙が開発を担当する。

**想定機能**:
1. 営業スキル体系の定義（スキルマップ、レベル別到達基準）
2. 育成プログラム管理（新人→一人前→リーダーの育成ステップ）
3. スキル評価・可視化（成長曲線、弱点検出）
4. AI活用（商談ロールプレイ分析、個別最適化された育成プラン提案）
5. ナレッジ共有（トップセールスのノウハウDB化、成功事例蓄積）

**開発期間**: 2年間（検証期間として）

**詳細要件**: 甲乙協議の上、別途仕様書を作成する。

```

```

## 第6条（秘密保持）

### 6.1 秘密情報の定義
```
本契約に関連して、甲乙が相手方から開示された情報のうち、
秘密である旨が明示された情報を「秘密情報」とする。
```

### 6.2 秘密保持義務
```
甲乙は、秘密情報を第三者に開示・漏洩せず、本契約の目的以外に使用しないものとする。
本義務は、契約終了後も5年間継続する。
```

### 6.3 例外
```
以下の情報は秘密情報から除外する：
- 開示時点で公知の情報
- 開示後に甲乙の責によらず公知となった情報
- 第三者から適法に取得した情報
- 独自に開発した情報
```

## 第8条（検収）

### 8.1 検収手続き
```
1. 乙は、本番環境デプロイ完了時に甲に対して成果物を納品する
2. 甲は、納品日から14日以内に検収を行う
3. 甲が検収完了の通知をした時点、または14日経過時点で検収完了とする
```

### 8.2 不合格の場合
```
甲が検収不合格と判断した場合、速やかに乙に通知する。
乙は、不合格理由を解消するための修正を無償で行う。
```

---

## 第9条（瑕疵担保）

### 9.1 瑕疵担保期間
```
検収完了日から6ヶ月間、乙は成果物の瑕疵について無償で修正を行う。
```

### 9.2 瑕疵の範囲
```
瑕疵とは、以下のいずれかに該当するものをいう：
- 仕様書に定めた機能が実装されていない
- 明らかなバグ・エラーが存在する
- セキュリティ上の重大な欠陥が存在する
```

---

## 第10条（保守サポート）

### 10.1 保守範囲
```
ランニング費用（月額12,500円）には、以下が含まれる：
- サーバー費用
- バグ修正
- セキュリティアップデート
- 軽微な機能改善
- 月次バックアップ
- 問い合わせ対応（営業日24時間以内）
```

### 10.2 保守範囲外
```
以下は別途見積となる：
- 新規機能の追加
- 大規模な仕様変更
- 第三者サービスとの連携
```

## 第11条（解約）

### 11.1 通常解約
```
甲または乙は、3ヶ月前までに書面で相手方に通知することにより、
本契約を解約することができる。
```

### 11.2 即時解約
```
以下の場合、相手方は直ちに本契約を解約できる：
- 相手方が本契約に違反し、催告後14日以内に是正されない場合
- 相手方が破産手続き開始の申立てを受けた場合

```

### 11.3 解約時のデータ引き渡し
```
解約時、乙は甲に対して以下を無償で提供する：
- 評価データの全件エクスポート（CSV/Excel形式）
- システム操作マニュアル
```
