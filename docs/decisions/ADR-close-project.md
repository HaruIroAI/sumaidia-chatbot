# ADR: プロジェクトクローズ

**日付**: 2026-03-02
**ステータス**: Accepted
**決定者**: 神子 勇一

---

## コンテキスト

- 60万円（税込）の入金が2026-02-27に確認された
- 書面契約は先方弁護士確認待ちのまま未締結で終了
- 保守契約（年15万円）は未締結
- システムは技術的に95%完成（残り5%は先方データ未提供のため完了不可）
- 石光社長から「6末〜7月に改めて連絡する」と返信あり（2026-02-27）

経緯の詳細は STATUS.md セクション2を参照。

## 決定

1. プロジェクトをクローズする
2. 保守契約は締結しない
3. 石光社長への最終返信はフラットな文面で送付する（過去に触れない）
4. Netlify上のチャットボットは稼働状態を維持するが、積極的な保守は行わない

## 理由

- 入金により金銭的な決着はついている
- 書面契約がないまま保守に入るリスクは取れない
- 先方から再連絡の意向があるため、関係は穏便に維持する

## 教訓（次回受注時の鉄則）

1. **書面契約→着手金入金→開発開始** の順を徹底する
2. クライアント起因の停滞は **2週間未回答→書面催促、1ヶ月→打ち切り通知**
3. 値下げ幅は **実工数の70%カバーを死守**（150万→60万は割安すぎた）
